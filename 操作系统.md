# 怎么杀死进程？
一般情况下，终止一个前台进程使用 Ctrl + C 就可以了。对于一个后台进程就须用 kill 命令来终止。  
我们会先使用 ps、top 等命令获得进程的 PID，然后使用 kill 命令来杀掉该进程。


# 线程，进程区别？
进程是资源分配的最小单位，线程是CPU调度的最小单位。
